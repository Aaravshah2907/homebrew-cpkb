# Aaravshah2907 Cpkb

## How do I install these formulae?

`brew install aaravshah2907/cpkb/<formula>`

Or `brew tap aaravshah2907/cpkb` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "aaravshah2907/cpkb"
brew "<formula>"
```
